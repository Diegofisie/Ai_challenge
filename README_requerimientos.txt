------------------------Solución para el código de Ai Challenge del grupo ElectroFiSIS-----------------------------------------------
- Requerimientos de Software:
Para la solución implementada se debe tener instalado Arduino.
Con las siguientes librerías:
-Pyserieal
-Numpy
-Rospy
-Random
para aplicar el controlador de inteligencia artificial se realizó la programación en Python 2.7.1 

-requerimientos de Hardware
Para la implementación del primer reto hay necesidad de un módulo Joystick para arduino, una display LCD y la estructura que funciona
como base para tanto el joystick como la pantalla. Todos estos componentes los tenemos nosotros implementados así que no hace falta que
 los organizadores del reto los provean. 

--------------------------------------------Comentarios extra-----------------------------------------------------------------
En la carpeta enviada hay 3 sub carpetas cada una con las carpetas de ROS creadas con las soluciones, la primera consiste en 
la solución para el controlador externo, la segunda para el controlador de AI y la tercera con el controlador multiplayer.
